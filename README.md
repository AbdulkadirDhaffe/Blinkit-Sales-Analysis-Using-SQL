# Blinkit Sales Analysis Using SQL

## Project Overview
<img width="798" height="488" alt="image" src="https://github.com/user-attachments/assets/3c7ea59b-4248-481d-af6b-d5352118aceb" />


This project analyses Blinkit's sales performance, customer preferences, and outlet performance using SQL Server. The objective was to transform raw transactional data into meaningful business insights that support strategic decision-making.

## Business Problem

Blinkit operates multiple outlet types across different locations and product categories. The company needs to understand which products, outlet types, and locations contribute most to revenue and customer satisfaction.

## Tools Used

* SQL Server Management Studio (SSMS)
* SQL
* CSV Dataset

## Project Workflow

1. Imported the raw Blinkit sales dataset into SQL Server.
2. Performed data cleaning and standardisation.
3. Calculated key performance indicators (KPIs).
4. Conducted sales and outlet performance analysis.
5. Generated business insights and recommendations.

## KPIs Analyzed

* Total Sales
* Average Sales
* Number of Items Sold
* Average Rating

## Analysis Areas

* Sales by Fat Content
* Sales by Item Type
* Sales by Outlet Establishment Year
* Sales by Outlet Size
* Sales by Outlet Location
* Sales by Outlet Type

## SQL Techniques Used

* Aggregate Functions
* GROUP BY
* ORDER BY
* CASE Statements
* Data Cleaning
* PIVOT Tables
* Window Functions
* Percentage Calculations

## Business Value

The project provides management with a clear understanding of product performance, outlet effectiveness, and customer preferences, enabling data-driven business decisions.

## Conclusion

This project demonstrates practical SQL skills in data cleaning, business analysis, KPI reporting, and insight generation using a real-world retail dataset.
